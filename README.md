# WebEscher
Escher-like periodic pavement editing

## Description
WebEscher est une application web interactive pour la manipulation de pavages du plan, inspirée de l'œuvre de M.C. Escher. L'application permet de créer des motifs de pavage en utilisant les 17 groupes de symétrie cristallographiques (wallpaper groups).

## Fonctionnalités
- **17 configurations de pavage** : Tous les groupes de symétrie cristallographiques du plan
- **Formes de base** : Carré, triangle, hexagone
- **Édition interactive** : Modifiez une forme et voyez toutes les instances se mettre à jour simultanément
- **Interface intuitive** : Faites glisser les points de contrôle pour transformer les formes

## Utilisation
1. Ouvrez `index.html` dans un navigateur web moderne
2. Sélectionnez un groupe de symétrie dans le menu déroulant
3. Choisissez une forme de base (carré, triangle ou hexagone)
4. Cliquez et faites glisser les points de contrôle pour modifier la forme
5. Observez comment toutes les instances du pavage se mettent à jour en temps réel

## Groupes de symétrie disponibles
- **Oblique** : p1, p2
- **Rectangulaire** : pm, pg, pmm, pmg, pgg, cmm
- **Rhombique** : cm
- **Carré** : p4, p4m, p4g
- **Hexagonal** : p3, p3m1, p31m, p6, p6m

## Technologies
- HTML5 Canvas pour le rendu graphique
- JavaScript vanilla (pas de dépendances)
- CSS3 pour le style moderne
